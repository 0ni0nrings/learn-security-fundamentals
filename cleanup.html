1. Home
  2. Clean up

Expected duration: ~5 minutes.

  1. Visit https://console.aws.amazon.com/iam/home#/users
  2. Select the check box next to your created user, click the Delete user button
  3. If you enabled a password policy, visit https://console.aws.amazon.com/iam/home#/account_settings
  4. Click the Delete password policy button
  5. If you created a IAM group, visit https://console.aws.amazon.com/iam/home#/groups
  6. Select the check box next to your created group, click the Delete group button
  7. Visit https://console.aws.amazon.com/s3/home?region=us-east-1
  8. Delete the bucket you created
  9. Visit https://console.aws.amazon.com/iam/home#/policies
  10. Select the check box next to your created policy, click the Delete button
  11. Visit https://console.aws.amazon.com/cloudtrail/home?region=us-east-1
  12. Delete the trail you created
  13. Visit https://console.aws.amazon.com/s3/home?region=us-east-1
  14. Delete the bucket, that CloudTrail created for you
