Create IAM User with Login Profile and Access Key

  1. Visit https://console.aws.amazon.com/iam/home#/users
  2. Click Add user button
  3. Set User name, enable Programmatic access to generate an Access Key, enable AWS Management Console access to create a Login Profile
  4. Manage the IAM Group later
  5. Review your input
  6. Copy and paste the sign-in link, your password and access key id & secret into a file on your machine for later use
  7. Use the sign-in link to test your IAM user in a different browser

Enable MFA

  1. Visit https://console.aws.amazon.com/iam/home#/users
  2. Click your user name
  3. Select Security Credentials tab, click the pencil icon next to Assigned MFA device
  7. Install Google Authenticator on your mobile device:
    • iTunes App Store
    • Google Play Store
  8. Open Google Authenticator and add a new token by scanning the QR code in your browser window
  9. Enter the first token and wait until a new token is generated. Enter the second token as well
  11. Use the sign-in link to test your IAM user in a different browser. THis time with MFA enabled.
