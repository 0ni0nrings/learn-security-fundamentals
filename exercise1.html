Exercise 1: IAM User

Expected duration: ~15 minutes.

Click on a task to get hints.

Task 1: Create IAM User with Login Profile and Access Key

  1. Visit https://console.aws.amazon.com/iam/home#/users
  2. Click Add user button
  3. Set User name, enable Programmatic access to generate an Access Key, enable AWS Management Console access to create a Login Profile
  4. Skip permissions as you will add them later
  5. Review your input
  6. Copy and paste the sign-in link, your password and access key id & secret into a file on your machine or download the CSV file for later use
  7. Use the sign-in link to test your IAM user in a different browser

Task 2: Enable MFA

  1. Visit https://console.aws.amazon.com/iam/home#/users
  2. Click your user name
  3. Select Security Credentials tab, click the pencil icon next to Assigned MFA device
  4. You are going to use the Google Authenticator app (a virtual device), choose A virtual MFA device
  5. You are asked if you have already installed a virtual device. You can acknowledged that because you will do it in the next step.
  6. Install Google Authenticator on your mobile device:
    • iTunes App Store
    • Google Play Store
  7. Now, you have enter two token generated with Google Authenticator.
  8. Open Google Authenticator and add a new token by scanning the QR code in your browser window
  9. Enter the first token and wait until a new token is generated. Enter the second token as well. Click Activate Virtual MFA.
  10. Success!
  11. Use the sign-in link to test your IAM user in a different browser. This time with MFA enabled.
