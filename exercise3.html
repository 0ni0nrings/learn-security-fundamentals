Enable CloudTrail

  1. Visit https://console.aws.amazon.com/cloudtrail/home?region=us-east-1
  3. Set a Trail name and a S3 Bucket (don't use the previous bucket name!). Stay with the defaults for the other fields.

Create dummy API calls

  1. Visit https://console.aws.amazon.com/ec2/v2/home?region=us-east-1
  2. Visit https://console.aws.amazon.com/iam/home
  3. Visit https://console.aws.amazon.com/rds/home?region=us-east-1
  4. Visit https://console.aws.amazon.com/s3/home?region=us-east-1

Look at results

  1. Visit https://console.aws.amazon.com/cloudtrail/home?region=us-east-1#/events
  2. It may take a few minutes until you see the first events!
  4. Explore the S3 bucket and look at at least one file that contains the events as well.
